The Project PerlMassFormatter.pl is based perl application that automatically reformat the input files in aligned columns, compress the file in size. By default signature is off for the capacity file purposes. This report is generated in the end of the finished process providing full information about your CPP update. The files are for security sake backuped into other files saved under the same path with your defined prefix in a file name. The new generated files should replace the old source files. This job can be started explicitelly by the same application, or joined to do it in one step (not recommended).

This tool can be started either separately or from ODBMApper application with a comfortable GUI.
